# [React search implementation ](https://react-search-implementation.netlify.app/)

## Description

React search implementation from scratch without libraries.

### Features:

- Caching
- Inline autocomplete
- Suggestions
- Last queries storing
- Fetch states
- Debouncing
- Arrow keys
- All in one hook

### Technologies used:

- React
- Next JS
- Typescript
- Tailwind
- Redux
